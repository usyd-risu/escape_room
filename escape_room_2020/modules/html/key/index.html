the sequences

the sequences were made to avoid groups that start at the same time all following each other around the library
each id has a different first letter and the arrays below deisgnate the order of the sequence.
for example, sequence one is abcd which means you play the game cake cat tea party caterpillar.
the rest of the challenges are connected so they always go gardeners, the tweedles and the finish.
below are the sequence orders:
